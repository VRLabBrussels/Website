Github pages for VR Lab Brussels.

Congratulation for being so curious. Normally participants to the lab and to its workshop should not have to look here. Still if you do feel free to check around. The most important part here is /jslibs/ that offers via https Javascript libraries that can be used directly in JSBin without creating mixed media warning in the browser. Maybe the repository will host more later on if the Wiki itself isn't enough.
